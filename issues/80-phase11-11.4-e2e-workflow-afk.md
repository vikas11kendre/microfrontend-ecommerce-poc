# 11.4 `e2e.yml` workflow

## What to build

A GitHub Actions workflow at `.github/workflows/e2e.yml` that runs the Playwright suite (10.6/10.7) on PRs to `main` and on push to PR branches. Steps:

1. Checkout
2. Setup Node
3. `npm ci`
4. `npx playwright install --with-deps`
5. `npm run build:all` (or rely on Playwright's `webServer` to build)
6. `npx playwright test`
7. Upload Playwright report as an artifact on failure

Refs: `sepc.md` §19.

## Acceptance criteria

- [ ] Workflow triggers on `pull_request` (to `main`) and `push` (PR branches)
- [ ] Installs Playwright browsers with `--with-deps`
- [ ] Runs the full Playwright suite
- [ ] Uploads `playwright-report/` as an artifact on failure
- [ ] Job times out at a sensible limit (e.g., 15 min)
