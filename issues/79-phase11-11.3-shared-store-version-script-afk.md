# 11.3 `scripts/check-shared-store-version.js`

## What to build

A Node script at `scripts/check-shared-store-version.js`. Walks every package's `package.json` for any dependency reference to `shared-store` (in `dependencies`, `devDependencies`, `peerDependencies`). Collects the version specifiers. Exits 1 if more than one distinct specifier is found.

This is the version-drift guard that pairs with the `singleton: true, strictVersion: true, requiredVersion: '^1.0.0'` MF config from 3.3 — together they make drift impossible to merge.

Refs: `sepc.md` §19.

## Acceptance criteria

- [ ] Script exists at `scripts/check-shared-store-version.js`
- [ ] Reads every `packages/*/package.json` and collects `shared-store` version specifiers
- [ ] Exits 0 when all references agree
- [ ] Exits 1 with a clear error message when references disagree
- [ ] Manually verifiable: temporarily change one package's `shared-store` dep to `^2.0.0` and run the script — must exit 1
