# 10.1 Install Vitest + RTL

## What to build

Configure Vitest at the repo root with `jsdom` environment and add per-package `test` scripts. Install `@testing-library/react`, `@testing-library/jest-dom`, and `vitest`.

Refs: `sepc.md` §15, §19.

## Acceptance criteria

- [ ] Root `vitest.config.ts` (or equivalent) configures `environment: 'jsdom'`
- [ ] Each package's `package.json` has a `test` script
- [ ] `@testing-library/react`, `@testing-library/jest-dom`, `vitest` installed at root
- [ ] `npm run test --workspaces` runs (empty test set is fine) without errors
