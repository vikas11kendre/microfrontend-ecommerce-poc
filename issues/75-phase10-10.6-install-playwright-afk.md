# 10.6 Install Playwright + configure `webServer`

## What to build

Install Playwright at root. Configure `playwright.config.ts` with the multi-app `webServer` pattern:
- Builds all 4 packages first (`npm run build:all`)
- Then starts each app via `npm start` on its assigned port
- Waits for each port to be reachable before running tests

Refs: `sepc.md` §19.

## Acceptance criteria

- [ ] `@playwright/test` installed at root
- [ ] `playwright.config.ts` defines a `webServer` array (one entry per app) OR uses `concurrently` via a single `command` — whichever the spec prefers
- [ ] `npx playwright test --list` enumerates test files without runtime errors
- [ ] `baseURL` set to `http://localhost:3000` (host)
